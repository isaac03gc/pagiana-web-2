Materiales

Los materiales requeridos para este trabajo son sencillos y muy faciles de encontrar en cualquier tienda cerca de ti

En la decoración de tejas, lo primero de todo, debes de tener unos conocimientos basicos de manualidades para poder hacer las figuras requeridas con pasta de modelar. Este tipo de decoración es poco conocida; pero desde hace poco tiempo se está difundiendo en las redes, porque desarrollas tu imaginación y puedes convertirlo en tu hobbit.

Los materiales basicos que necesitas para hacer esta teja son: una teja de barro de un tamaño medio, pasta para modelar de ceramica blanca, pasta para modelar de arcilla, pegamento rápido, cola blanca, pinturas acrílicas para manualidades, brochas de diferentes tamaños, cuchillo afilado, cutters estilo bisturí, tijeras, regla, pistola de silicona transparente, paleta para mezclar pinturas, lapiz, papel semiduro para las plantillas, paño con encaje o gasa, barrita de tinta para boligrafo y rodillo para aplastar la pasta de moledar.
